OpenResponses

A powerful native SwiftUI playground for exploring the full range of OpenAI's API capabilities. Features comprehensive tool integration including computer use, web search, code interpreter, image generation, file search, and extensive Model Context Protocol (MCP) support.

Support Privacy Policy View on GitHub

Key Features

  • Multi-Model Support: Access GPT-5, GPT-4.1, O-series reasoning models, and specialized computer-use models.
  • Enhanced Streaming: Real-time responses with granular status updates for AI thinking, web searching, and tool execution.
  • Computer Use: Production-ready browser automation with safety approval workflows.
  • Web Search: Access up-to-date information with standard and preview search capabilities.
  • Code Interpreter: Execute Python in secure sandboxed environments with file preloading and rich artifact display.
  • Image Generation: Create images from text prompts with real-time streaming previews.
  • File Search: Search across multiple vector stores with support for 43+ file types.
  • MCP Integration: Connect to Model Context Protocol servers (GitHub, Notion, Slack, etc.).
  • Advanced Controls: Fine-tune temperature, tokens, penalties, reasoning effort, and more.
  • Developer Tools: API inspector, debug console, analytics dashboard, and comprehensive logging.
