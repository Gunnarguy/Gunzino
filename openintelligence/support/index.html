Frequently Asked Questions

Is sign-in required?

No. OpenIntelligence does not require an account to use core functionality.

Where is my content stored?

Documents and knowledge containers are stored locally on your device. You can delete documents and containers within the app.

Does OpenIntelligence send my documents to the cloud?

By default, processing is performed on-device, including the core Apple Intelligence and agentic retrieval path. Cloud pathways are optional and require explicit user consent. When enabled, only the active query and selected retrieved context snippets are transmitted as needed to generate a response.

How do I delete my data?

You can delete individual documents and entire knowledge containers within the app. You can also remove stored credentials (for example, API keys) from Settings. Uninstalling the app removes local application storage.

App Review Notes

Core functionality is available without login. Optional external integrations require user-provided credentials. Any cloud-bound requests require explicit consent and are recorded for transparency.
