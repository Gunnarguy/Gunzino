OpenCone

On-Device Retrieval Augmented Generation for iOS. Process PDFs, DOCX, TXT files, and images via OCR — convert them into vector embeddings with OpenAI, store them in Pinecone, and perform intelligent semantic searches to get AI-powered answers.

Support Privacy Policy View on GitHub

Key Features

  • Multi-Format Document Import: Upload and process PDFs, DOCX, TXT files, and images with on-device OCR.
  • OpenAI Embeddings: Generate high-dimensional vector embeddings using configurable models (e.g., text-embedding-3-large).
  • Pinecone Integration: Full lifecycle management for serverless vector indexes — create, list, select, and manage namespaces.
  • Semantic Search & RAG: Ask natural language questions and receive AI-generated answers grounded in your documents.
  • On-Device Processing: Text extraction runs locally using PDFKit, Vision OCR, and native Apple utilities.
  • Custom Design System: Bespoke OCDesignSystem with themeable components.
  • Detailed Statistics: View processing timings, token counts, and chunk distributions.
  • Guided Setup: User-friendly WelcomeView for initial API key configuration.
